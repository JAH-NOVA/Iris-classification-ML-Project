# 🌸 Iris Flower Classification - ML Project

## 📌 Overview
This project focuses on classifying Iris flower species using Machine Learning algorithms. It uses features like sepal length, sepal width, petal length, and petal width.

## 🚀 Project Highlights
- Performed data cleaning and preprocessing
- Trained multiple ML models
- Compared model performance

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## 📊 Models Used
- Support Vector Machine (SVM)
- Logistic Regression
- Decision Tree

## 📈 Results
- Achieved up to **93% accuracy**
- Evaluated using:
  - Precision
  - Recall
  - F1-score

## 🔍 Workflow
1. Data Loading  
2. Data Cleaning  
3. Feature Selection  
4. Model Training  
5. Model Evaluation  

## 💡 Key Learnings
- Importance of data preprocessing  
- Comparing multiple models  
- Understanding evaluation metrics
